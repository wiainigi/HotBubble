Features

Guide

使用说明

快速上手，即刻体验

1

下载解压

下载压缩包后解压，运行 HotBubble.exe 即可启动。

2

系统托盘

软件运行后会在任务栏右下角显示图标，在图标上右键可以打开菜单。

3

退出软件

从右下角图标右键菜单选择「退出」即可关闭软件。

4

添加快捷键配置

将配置文件添加到目录下的 keyboards 文件夹，文件名需与软件进程名称一致。

5

编辑配置文件

快捷键配置文件可自行编辑，根据需要删除或添加快捷键条目。

Configuration

配置文件说明

配置文件存放在 keyboards/软件进程名.ini

keyboards/Windows.ini 
[Info]
# 软件名称（如果软件名称和软件说明都省略则默认显示软件进程名）
Name=Windows
# 软件说明
Description=系统默认

[Keys]
Win=打开/关闭开始菜单
# 快捷键使用 + 号连接
Win+D=显示桌面
Win+E=打开文件资源管理器
