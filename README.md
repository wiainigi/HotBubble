# 热键泡泡（HotBubble）

热键泡泡是一款能够在按下Ctrl、Alt、Shift、Win修饰键时弹出快捷键提示的小工具，使用C++进行AI Coding编写完成。

> 推荐给对快捷键不熟悉的初学者使用，能够快速熟悉windows或软件快捷键

## 快捷键类型

- 系统快捷键：windows快捷键
- 软件快捷键：当前聚焦的窗口对应的软件快捷键

## 软件特色

- 快捷键分类：支持显示两种类型的快捷键，并且可以通过软件设置自由开关显示
- 按键标记：被按下的按键会进行颜色标记
- 自定义主题：可以自选背景颜色、透明度、字体大小、颜色等
- 超小程序包：软件安装包超小，不到1MB
- 超小占用：程序挂后台占用仅2MB左右内存
- 自定义配置文件：可以根据自己使用的软件自定义添加配置文件
- 位置识别：根据鼠标编辑位置自动切换热键提示界面显示位置，不盖住操作界面

## 使用说明

- 下载压缩包后解压运行 **HotBubble.exe** 即可
- 软件运行后会在任务栏右下角显示图标，在图标上右键可以打开菜单
- 退出软件需要从右下角图标右键菜单选择退出
- 添加对应软件的快捷键配置直接将配置文件添加到目录下的keyboards即可（文件名称和软件进程名称一致）
- 快捷键配置文件可自行编辑（比如不需要那么多快捷键可以自行删除或添加）

## 配置文件说明

> 配置文件存放在根目录的keyboards/软件进程名.ini

配置文件格式如下

```ini
[Info]
# 软件名称（如果软件名称和软件说明都省略则默认显示软件进程名）
Name=Windows
# 软件说明（如果软件名称和软件说明都省略则默认显示软件进程名）
Description=系统默认

[Keys]
Win=打开/关闭开始菜单
# 快捷键使用+号连接，严格按以下格式编写
Win+D=显示桌面
Win+E=打开文件资源管理器
# 添加更多的快捷键提示
```
